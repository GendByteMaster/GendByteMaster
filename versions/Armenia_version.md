# GendByteMaster - Հենդրիկ Մինասարյան

**Հենդրիկ Մինասարյան**

Skyend Technologies-ի հիմնադիր։ Ծրագրավորող։ Խնդիրներ լուծող։ Error-ը իմ երկրորդ անունն է, բայց սովորելն ինձ համար միշտ առաջին տեղում է։

> "Ծրագրավորողի երկրորդ անունը Error-ն է։ Բոլոր մարդիկ սխալվում են, և ես բացառություն չեմ։ Ես միշտ փորձում եմ սովորել իմ սխալներից և այդ փորձը վերածել օգտակար ու արդյունավետ լուծումների։"

## Իմ մասին

### Իմ ճանապարհը

Ես ինքնուս ծրագրավորող եմ, որը սիրում է հաղթահարել սահմանափակումները։ Ուղեղային կաթվածը ամեն օր բերում է խոսքի և շարժման դժվարություններ, բայց դա երբեք չի կանգնեցրել ինձ ծրագրեր կառուցելու ճանապարհին։ Ընդհակառակը, դա ինձ մղում է ստեղծելու արդյունավետ, մատչելի և կայուն թվային արտադրանքներ։

### Կոդից դուրս

Երբ ես debug չեմ անում, սովորաբար կորում եմ Հայաո Միյաձակիի աշխարհներում, անիմեի պատմություններում կամ նոր երաժշտության մեջ։ Կոդից դուրս ստեղծագործականությունը ինձ օգնում է ավելի հնարամիտ մնալ նաև ծրագրավորման մեջ։

`Անիմեի սիրահար` `Երաժշտության սիրահար` `Ինքնուս`

_Մարտ 2025 - Ապրիլ 2025_

- Կիրառել եմ DAG մոտեցումը՝ ավելի հստակ կառուցվածքի և կազմակերպված մշակման համար։

- Ներդրել եմ նոր server-side ֆիչաներ հարթակի համար։

- Միավորել եմ արտադրանքի տեսլականը և գործնական համակարգային ճարտարապետությունը։

## Կարեւոր նախագծեր

### CellNexus - Մշակման փուլում

CellNexus-ը մոդուլային peer-to-peer DAG ցանց է typed cellular automata data-ի հոսքային փոխանակման և պահպանման համար։ Այն միավորում է ստուգելի DAG core, persistent storage և libp2p overlay, որպեսզի node-երը կարողանան փոխանակել state-ը real time-ում։

**Tech:** Rust, DAG architecture, API, WebSocket, MongoDB, Redis

### MysticPulse - Open Source

Փորձարարական DAG բլոկչեյն։

**Tech:** Rust, MYSTICETI consensus, Ed25519 cryptography, WebSocket, MongoDB, Redis

## Ընտրված նախագծեր

### OHLC AI

AI-powered առեւտրային վերլուծություն real-time տվյալների մշակմամբ և predictive algorithm-ներով։

**Tech:** Python, TensorFlow, FastAPI, Docker

### Gridlock

Ռազմավարական բազմախաղացող խաղ՝ զարգացած մարտական համակարգերով և real-time multiplayer-ով։

**Tech:** TypeScript, React, Node.js, WebSocket

### FlowForge - Visual Workflow Studio

FlowForge-ը HIG-inspired canvas է automation pipeline-ների նախատիպավորման համար։ Կառուցեք flow-եր վերաօգտագործվող node-երով, սիմուլացրեք կատարումը live feedback-ով և արտահանեք JSON կամ YAML անմիջապես բրաուզերից։

**Tech:** TypeScript, React, React Flow, WebSocket

### [Orchestration System](https://github.com/GendByteMaster/orchestration-system)

Orchestration System-ը աշխատանքային հոսքերի օրկեստրացիայի նախագիծ է՝ համակարգի բաղադրիչների միջև առաջադրանքները, կատարման հոսքը և ավտոմատացման տրամաբանությունը համակարգելու համար։

**Tech:** Orchestration, Automation, Workflow Systems

### Zoltraak - Secure Exchange on CellNexus

Zoltraak-ը ապահովում է գաղտնի ֆայլերի փոխանակում CellNexus հարթակի վրա։ Ստեկը միավորում է FastAPI backend, React frontend և ապագա mini-app tooling-ը CellNexus ծառայությունների հետ՝ կոդավորված տվյալների երթուղավորման, audit-ի և telemetry-ի համար։

**Tech:** Python, FastAPI, React, WebSocket, CellNexus

### Slotio

Կրոսպլատֆորմային mobile հավելված մասնավոր մասնագետների համար՝ բարբերներ, կոսմետոլոգներ, դասատուներ և լուսանկարիչներ։ Մասնագետները մուտք են գործում մեկ սեղմումով Telegram OAuth-ով, կարգավորում են պրոֆիլն ու ժամանակացույցը, կիսվում հղումով և թույլ են տալիս հաճախորդներին ամրագրել առանց ավելորդ նամակագրության։

**Tech:** Flutter, iOS, Android, Telegram OAuth

### Rezervo

Rezervo-ն ժամանակակից հարթակ է վարպետների համար, որոնք ուզում են ավելի քիչ նամակագրություն և ավելի շատ իրական աշխատանք։ Հաճախորդները գտնում են վարպետին որոնման կամ ուղիղ հղման միջոցով և ամրագրվում են ընդամենը 4 քայլով։ Ամեն ինչ կառուցված է Next.js 16-ի և Supabase-ի վրա։

**Tech:** Next.js 16, Supabase

### AetherShield

Տվյալների կոդավորման նորարարական համակարգ՝ ոգեշնչված անիմե կոնցեպտներով և ուժեղացված արհեստական բանականությամբ։ Բաղադրիչները ներառում են AetherCore (Rust), AetherNexus (Python), AetherMind, AetherOracle, AetherVault և AetherGate։

**Tech:** Rust, Python, FastAPI, AI, Claude

## Ընտրված ապրանք: FlowForge - Visual Workflow Studio

Կենտրոնացած տեսողական ստուդիա ավտոմատացման հոսքերը նախագծելու, փորձարկելու և մշակման փոխանցելուց առաջ հստակեցնելու համար։

### Ինչ է անում FlowForge-ը

FlowForge-ը տալիս է node-ային canvas՝ ավտոմատացման տրամաբանությունը տեսողական կառուցելու, հոսքերի վարքը կենդանի արձագանքով ստուգելու և ինժեներական փոխանցման կամ ներքին նախատիպավորման համար մաքուր կառուցվածքային արդյունք արտահանելու համար։

### Հիմնական հնարավորություններ

- Տեսողական node-ային canvas բարդ ավտոմատացման հոսքերի համար
- Վերաօգտագործվող բլոկներ՝ հոսքերի կառուցումը արագացնելու համար
- Կենդանի արձագանք՝ տրամաբանությունը փորձարկելիս
- JSON կամ YAML արտահանում անմիջապես բրաուզերից

### Հարմար է

- Ավտոմատացման հոսքեր նախագծող product թիմերի համար
