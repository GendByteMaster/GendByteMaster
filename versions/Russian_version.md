## Ключевые проекты

### CellNexus - В разработке

CellNexus — это модульная peer-to-peer DAG-сеть для стриминга и хранения типизированных данных cellular automata. Она объединяет верифицируемое DAG-ядро, persistent storage и libp2p-слой, чтобы узлы могли обмениваться состоянием в реальном времени.

**Tech:** Rust, DAG architecture, API, WebSocket, MongoDB, Redis

Zoltraak обеспечивает конфиденциальный обмен файлами поверх платформы CellNexus. Стек объединяет FastAPI backend, React frontend и будущий mini-app tooling с сервисами CellNexus для маршрутизации зашифрованных данных, аудита и телеметрии.

**Tech:** Flutter, iOS, Android, Telegram OAuth

### Rezervo

Rezervo — это современная платформа для мастеров, которые хотят меньше переписываться и больше работать. Клиенты находят мастера через поиск или по ссылке, а затем записываются всего за 4 шага. Всё построено на Next.js 16 и Supabase.

**Tech:** Next.js 16, Supabase

### AetherShield

Инновационная система шифрования данных, вдохновлённая аниме-концептами и усиленная искусственным интеллектом. Компоненты: AetherCore (Rust), AetherNexus (Python), AetherMind, AetherOracle, AetherVault и AetherGate.

**Tech:** Rust, Python, FastAPI, AI, Claude

## Избранный продукт: FlowForge - Visual Workflow Studio

Сфокусированная визуальная студия для проектирования, проверки и доработки automation flow до передачи в разработку.

### Что делает FlowForge

FlowForge даёт node-based canvas для визуальной сборки automation-логики, проверки поведения flow с live feedback и экспорта чистого структурированного результата для handoff в разработку или внутреннего прототипирования.

### Ключевые возможности

- Визуальный canvas с node для сложных automation-сценариев
- Переиспользуемые блоки для более быстрой сборки flow
- Live feedback при тестировании логики
- Экспорт в JSON или YAML прямо из браузера

### Подходит для

- Продуктовых команд, проектирующих automation workflows
- Основателей, которые валидируют внутренние инструменты до реализации
- Студий, которым нужен быстрый переход от прототипа к техническому описанию
