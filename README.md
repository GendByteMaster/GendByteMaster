## Featured Product: FlowForge - Visual Workflow Studio

A focused visual studio for mapping, testing, and refining automation flows before they reach production.

### What FlowForge does

FlowForge gives you a node-based canvas for building automation logic visually, checking flow behavior with live feedback, and exporting clean structured output for engineering handoff or internal prototyping.

### Core capabilities

- Visual node canvas for complex automation paths
